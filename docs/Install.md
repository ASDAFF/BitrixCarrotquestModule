### Установка:
1. Установить сайт (Битрикс управление сайтом + веб окружение) http://www.1c-bitrix.ru/download/cms.php
2. Поместить в папку /bitrix/www/bitrix/modules
3. Зайти за админа и перейти в администрирование сайта
4. Выбрать в меню "Настройки" -> "Модули". Нажать установить.
5. Ввести ключи, они сохранятся в настройках модуля.
6. После успешной устнаовке в "Настройках модулей" будет доступны настройки модуля carrotquest.
