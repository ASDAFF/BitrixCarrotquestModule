### Установка из Marketplace
1. Зайти на своем сайте под учетную запись администратора
2. Перейти на Marketplace, ввести ссылку на свой сайт, нажать установить.
3. Откроется предложение загрузить модуль в административной панели Вашего сайта.
4. Нажать "загрузить".
5. Прочитать и принять лицензионое соглашение (установив галочку).
6. Нажать "установить".
7. Следовать подсказкам мастера установки.

### Ручная установка:
1. Установить сайт (Битрикс управление сайтом + веб окружение) http://www.1c-bitrix.ru/download/cms.php
2. Зайти за админа и перейти в администрирование сайта
3. Поместить в папку /bitrix/www/bitrix/modules. 
4. Выбрать в меню "Marketplace" -> "Установленные решения". В списке должен быть модуль Carrot quest.
5. Открыть меню модуля (3 горизонтальные полоски) и нажать установить. Следовать подсказкам мастера установки.

### Мастер установки
1. Ввести ключи, они сохранятся в настройках модуля. Ключи выдаются при заключении соглашения с компанией [Carrot quest](http://carrotquest.ru).
2. Выбрать шаблоны сайтов (из папки /bitrix/templates/), в которые будут встроены сервисы Carrot quest. Копии исходных шаблонов будут сохранены в папке /bitrix/modules/carrotquest.analytics/templatesBackup
3. Появится окно, информирующее о результатах установки.
 Для получения помощи по установке, вы можете ввести свой Email в форму обратной связи и нажать "Отправить". Мы обязательно с Вами свяжемся.

### Настройки модуля
После успешной установки в меню "Настройки" -> "Настройки модулей" административной панели будут доступны настройки модуля Carrot quest. С помощью них Вы можете:
1. Управлять отправкой системных событий магазина;
2. Включить/выключить бонусную систему;
3. Изменить ключи для связи с сервером Carrot quest.
4. Изменить список шаблонов сайтов, в которые встроен Carrot quest.
